# Social Media Dashboard

A MERN stack based social media dashboard with real-time messaging and analytics.

## Features
- User authentication
- Create posts, like and comment
- Real-time chat using Socket.IO
- Analytics dashboard
- Redis notifications

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Socket.IO
- Redis

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
